# Sources and provenance

## Interaction research

This project studies publicly observable interaction patterns from OpenAI Codex Desktop and other coding-agent products.

All components, visual assets, styles, design tokens, and implementation code in this repository are independently created. No code or assets extracted from the Codex Desktop `app.asar` bundle are included in or used as implementation source for this repository.
